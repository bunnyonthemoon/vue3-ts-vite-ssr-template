## Требования

1. `Node.js ^12.13 || >=14`
2. `npm`
3. `pm2`

## Команды

### Установка зависимостей

```sh
yarn
```

### Запуска сервера

```sh
pm2 start
```

### Запуска сервера в Dev режмие

```sh
pm2 start —env development
```

### Остановка сервера

```sh
pm2 stop hr-server
```

### Dev режим для фронтенда

```sh
yarn dev
```

### Создание билда фронтенда

```sh
yarn build
```
